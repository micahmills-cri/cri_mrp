## Design System

### Color Palette

**Primary Colors:**
```
Primary Blue:      #2563EB  (Buttons, links, active states)
Primary Dark:      #1E40AF  (Hover states)
Primary Light:     #DBEAFE  (Backgrounds)
```

**Status Colors:**
```
Success Green:     #10B981  (Completed, success messages)
Warning Yellow:    #F59E0B  (In Progress, warnings)
Danger Red:        #EF4444  (Holds, errors, urgent priority)
Info Blue:         #3B82F6  (Released, info messages)
Gray:              #6B7280  (Disabled, secondary text)
```

**Priority Colors:**
```
LOW:               #9CA3AF  (Gray)
NORMAL:            #3B82F6  (Blue)
HIGH:              #F59E0B  (Orange)
URGENT:            #EF4444  (Red)
```

**Status Badge Colors:**
```
PENDING_REVIEW:    #F59E0B  (Orange)
RELEASED:          #3B82F6  (Blue)
IN_PROGRESS:       #10B981  (Green)
ON_HOLD:           #EF4444  (Red)
COMPLETED:         #10B981  (Green)
CANCELLED:         #6B7280  (Gray)
```

### Typography

**Font Family:** Inter (sans-serif)

**Sizes:**
```
Heading 1:         32px (2rem) - Bold
Heading 2:         24px (1.5rem) - Semibold
Heading 3:         20px (1.25rem) - Semibold
Body:              16px (1rem) - Regular
Small:             14px (0.875rem) - Regular
Tiny:              12px (0.75rem) - Regular
```

**Operator Console (Tablet):**
```
Heading 1:         40px (2.5rem) - Bold
Heading 2:         32px (2rem) - Semibold
Body:              20px (1.25rem) - Regular
Buttons:           24px (1.5rem) - Semibold
```

### Spacing

```
xs:  4px
sm:  8px
md:  16px
lg:  24px
xl:  32px
2xl: 48px
```

### Touch Targets (Operator Console)


## Responsive Breakpoints

```
Mobile:     < 640px  (Operator console primary target: tablets 768px+)
Tablet:     640px - 1024px
Desktop:    > 1024px
```

**Operator Console:**
- Optimized for tablets: 768px - 1024px (landscape orientation)
- Still usable on phone in portrait for emergency access

**Admin/Supervisor:**
- Optimized for desktop: > 1024px
- Sidebar collapses to hamburger menu on tablet/mobile

---

## Accessibility

- **Color Contrast:** WCAG 2.1 AA standard (4.5:1 for text)
- **Touch Targets:** Minimum 44px Ã— 44px (operator console: 56px+)
- **Keyboard Navigation:** All interactive elements focusable via Tab
- **Screen Reader:** Proper ARIA labels on all components
- **Focus Indicators:** Clear blue outline on focused elements

---

## Animation & Transitions

- **Page Transitions:** Fade in (200ms)
- **Modal Open/Close:** Scale + fade (250ms)
- **Button Hover:** Color transition (150ms)
- **Sidebar Toggle:** Slide (300ms ease-in-out)
- **Toast Notifications:** Slide in from top (200ms)
- **Loading States:** Spinner fade in after 300ms (avoid flash on fast loads)

**Keep animations subtle - performance and usability over flashiness.**
